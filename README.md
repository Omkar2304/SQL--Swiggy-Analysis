<h1 align="center">🍔 SQL: Swiggy Data Analysis</h1>

<h2>📌 Project Overview</h2>
<p>
This project presents a comprehensive <b>Swiggy Data Analysis</b> using SQL to analyze restaurant data, 
customer preferences, food categories, pricing, ratings, and city-wise restaurant performance. 
The project demonstrates how SQL queries can be used to extract meaningful business insights from 
food delivery platform datasets and support data-driven decision-making.
</p>

<h2>🎯 Problem Statement</h2>
<p>
Food delivery platforms generate massive amounts of data daily, making manual analysis difficult 
and inefficient. Businesses need an efficient system to analyze restaurant performance, customer trends, 
pricing strategies, and ratings. This project provides SQL-based analysis to uncover insights about 
restaurant operations, customer preferences, and food delivery trends.
</p>
